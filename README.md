# Descripción del proyecto

Este portafolio recopila una serie de actividades interactivas diseñadas para reforzar los conceptos fundamentales de álgebra. Cada actividad aborda un tema específico, desde números complejos hasta ecuaciones de primer grado, proporcionando herramientas prácticas para la comprensión y aplicación de los principios algebraicos. A través de este portafolio, se busca demostrar el dominio de los temas tratados en clase mediante la creación de recursos educativos accesibles y funcionales.

---

# Instrucciones para ejecutar el portafolio

1. Descargar o clonar el repositorio completo del portafolio.

```bash
git clone https://github.com/bylev/Portafolio-de-evidencias.git
```

2. Abrir el archivo `index.html` en un navegador web moderno (como Chrome, Firefox o Edge).

3. Navegar por las diferentes actividades utilizando los enlaces proporcionados en la página principal.

4. Interactuar con las herramientas y ejercicios disponibles en cada actividad para practicar y reforzar los conceptos aprendidos.

---

# Descripcion de las actividades incluidas

- **Actividad 2: Números Complejos** - Herramienta interactiva para visualizar y operar con números complejos en el plano complejo.

- **Actividad 4: Conversión de Forma Polar a Exponencial** - Calculadora para convertir números complejos de forma polar a forma exponencial.

- **Actividad 6: Expresiones Algebraicas** - Generador de expresiones algebraicas con simplificación automática.

- **Actividad 8: Calculadora de Operaciones Algebraicas** - Calculadora para realizar operaciones básicas con expresiones algebraicas.

- **Actividad 10: Productos Notables** - Herramienta para identificar y aplicar productos notables en expresiones algebraicas.

- **Actividad 12: Factorización de Expresiones Algebraicas** - Calculadora para factorizar expresiones algebraicas.

- **Actividad 14: Ecuaciones de Primer Grado** - Solucionador interacto de ecuaciones de primer grado con pasos detallados.
